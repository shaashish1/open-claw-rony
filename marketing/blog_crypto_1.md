## Key SEBI Rules for Algo Trading in India (2025)

Before you pick your platform, understand the regulatory landscape:

- **Exchange Approval Required:** Every algo strategy must be approved by NSE/BSE before live deployment
- **Unique Algo ID Tagging:** Each order must carry an exchange-assigned Algo ID
- **Static IP Mandatory:** API trading requires a whitelisted static IP address
- **TOPS Limit:** Orders exceeding 10/second require formal exchange registration
- **Two-Factor Authentication:** All algo platforms must implement 2FA
- **Broker as Custodian:** Your broker is legally responsible for your algo's compliance

Choose platforms that have built-in SEBI compliance workflows (AlgoTest, uTrade Algos, Tradetron all do this well).
