### Known Issues / Lessons Learned
- `ps aux` not available inside `openalgo-web` container — use logs or `STRATEGY_CONFIGS` check
- PowerShell heredoc syntax `@'...'@` fails with complex Python/bash inside SSH — use `scp` to copy scripts, then `ssh ... bash /tmp/script.sh`
- Windows `curl` = PowerShell alias (no `-c`/`-b`) — always SSH to VPS for real curl
- CRLF line endings from Windows cause `\r` in paths on Linux — watch for `tail: cannot open 'file'\r'`
- `2>/dev/null` in PowerShell heredoc causes "C:\dev\null not found" — avoid in PS strings
- Deploy cron reverts manual VPS edits — always commit to git first, then deploy
- FastAPI route order matters: catch-all `/{full_path:path}` must have `/ops` exception INSIDE it, not as separate route before it (lifespan context issue)
